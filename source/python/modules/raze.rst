Raze
****

Links
=====

- http://pypi.python.org/pypi/raze/
- http://docs.icecave.com.au/raze/

Install
=======

::

  pip install raze

Sample
======

``phonetic`` - NATO phonetic alphabet utilities:

::

  >>> from raze.phonetic import PhoneticDictionary
  >>> pd = PhoneticDictionary()
  >>> pd.spell('Pat46')
  'PAPA-alfa-tango-FOWER-SIX'
