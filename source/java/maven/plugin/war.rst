Name
====

To set the name of the ``war`` file:

::

  <build>
    <plugins>
      <plugin>
        <artifactId>maven-war-plugin</artifactId>
        <configuration>
          <warName>sample-servlet</warName>
        </configuration>
      </plugin>
    </plugins>
  </build>
