Ubuntu
======

Console
-------

*Ctrl*, *Alt*, *Function Key*

Note: Don't be surprised if the Gnome desktop is not on
*Ctrl*, *Alt*, *F1*.

Re-start Desktop
----------------

*Ctrl*, *Alt*, *Back-space*.

If *Ctrl*, *Alt*, *Back-space* is disabled, type:

::

  sudo /etc/init.d/gdm restart

Version
=======

::

  gnome-shell --version
