dwm - shortcut keys
*******************

Create a new terminal:
  *Shift*, *Alt*, *Enter*

Toggle window:
  *Alt*, *Enter*

Menu (``dmenu``):
  *Alt*, *P* (only if it is installed)...

Move to another window:
  *Alt*, *J* and *Alt*, *K*

Quit:
  *Shift*, *Alt*, *Q*

Tags
====

Switch to another tag:
  *Alt*, *2* or *Alt*, *3* etc...
