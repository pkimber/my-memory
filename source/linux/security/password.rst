Passwords
*********

Automated Password Generator
============================

Install
-------

::

  sudo apt-get install apg

Usage
-----

::

  $ apg

  Please enter some random data (only first 8 are significant)
  (eg. your old password):>
  Aroishruk5 (Ar-oi-shruk-FIVE)
  EcMivvubaj0 (Ec-Miv-vu-baj-ZERO)
