Folder
------

`How to copy a directory/folder`_.

To copy a folder, use the ``xcopy`` command:

::

  XCOPY E:\releases S:\backups\releases /E

Create
======

To create an empty file in DOS:

::

  touch c:\temp\filename.txt

Delete
======

Recursive
---------

::

  del /s .classpath
