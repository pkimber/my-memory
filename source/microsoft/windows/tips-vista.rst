telnet
======

`Where is the Telnet client in Windows Vista?`_

- *Control Panel*, *Programs*, *Turn Windows features on or off*.
- Tick *Telnet Client*.
- Click *OK*.

Windows Search
==============

If you don't use the *Windows Search* service, then turn it off.  *Services*,
*Windows Search*, *Stop* and *Disable*.
