Download
========

- Download ``latest.zip`` from http://markitup.jaysalvat.com/downloads/

Install
=======

- Extract to your site media folder:

  ::

    cd media
    unzip ~/download/markitup/latest.zip

- Move some files around...

  ::

    mv latest/* .
    rm -r latest/
    mv jquery-1.3.2.min.js jquery.js
