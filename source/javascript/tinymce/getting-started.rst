Getting Started
***************

Install
=======

Unzip the folder ``tinymce/jscripts/tiny_mce/``:

Linux
-----

::

  unzip ~/download/tinymce/tinymce_3_3rc1.zip tinymce/jscripts/tiny_mce/*
  mv tinymce/jscripts/tiny_mce/ .
  rm -r tinymce/

Windows
-------

::

  C:\projects\my_webapp\src\main\webapp\javascript\tiny_mce\
