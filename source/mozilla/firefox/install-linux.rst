Ubuntu

`Firefox 4 PPA for Ubuntu 10.04 and 10.10 users`_:

::

  sudo add-apt-repository ppa:mozillateam/firefox-stable
  sudo apt-get update && sudo apt-get upgrade
  sudo apt-get install firefox ubufox
