Attributes
**********

From `Custom Attributes in HTML 5`_

In HTML 5, you define custom attributes using the ``data`` attribute.  The
exact format is ``data-*`` e.g:

.. code-block:: html

  <div id="mydiv" data-brand="toyota" data-model="prius">
