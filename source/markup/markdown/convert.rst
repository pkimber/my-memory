Markdown to HTML
================

markdown (:doc:`../../python/modules/markdown`):

  mkvirtualenv markdown
  pip install markdown

  workon markdown
  markdown_py foo.md > foo.html

markdown2::

  mkvirtualenv markdown
  pip install markdown2

  workon markdown
  markdown2 foo.md > foo.html
